## Project Overview

The Crypto Lab is designed to provide users with up-to-date information on cryptocurrency prices and related news. The project leverages APIs to fetch real-time data and presents it in a visually appealing and interactive manner.

## Features

- Real-time cryptocurrency price tracking
- News updates related to cryptocurrencies
- Caching mechanism to optimize performance
- Responsive design for various devices
- Parallax scrolling effect for enhanced visual appeal

## Technologies and Libraries

- **React.js:** Front-end library for building user interfaces
- **Next.js:** Framework for server-side rendering and static site generation
- **axios:** Promise-based HTTP client for making API requests
- **React Slick:** Carousel library for displaying news in a slider
- **NewsAPI:** Free API for fetching news articles about cryptocurrencies
- **CoinGecko API:** Free API for fetching cryptocurrency data
- **CSS Modules:** Scoped CSS to avoid style conflicts

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/the-crypto-lab.git
   ```

2. Navigate to the project directory:

   ```bash
   cd the-crypto-lab
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000`.

## Usage

- **Home Page:** View a list of cryptocurrencies with their current prices.
- **Search:** Use the search bar to find specific cryptocurrencies.
- **Details:** Click on a cryptocurrency to view detailed information and related news.
